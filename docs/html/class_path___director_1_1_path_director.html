Detailed Description

Task-level controller that sequences Romi through predefined path states.

This class wraps the motion logic for each segment of the course, sharing data with other tasks via task_share queues and using closed-loop controllers for heading and line tracking. States are defined as integers and executed inside the generator run().

Generator implementing the state machine for path execution.
