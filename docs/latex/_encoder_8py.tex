\doxysubsection{Detailed Description}
This file implements a quadrature encoder interface for the Romi robot. 

The encoder uses a hardware timer in {\ttfamily Timer.\+ENC\+\_\+\+AB} mode to measure wheel rotation and compute velocity. The class inherits from {\ttfamily Sensor} so timing and velocity updates integrate cleanly with the controllers.

Features\+:
\begin{DoxyItemize}
\item Reads incremental wheel motion from a hardware timer
\item Converts tick differences into wheel radians
\item Accumulates total wheel displacement
\item Computes velocity in mm/s (using {\ttfamily dt} from Sensor)
\item Supports resetting via {\ttfamily zero()} 
\end{DoxyItemize}
