\doxysubsection{Detailed Description}
Reads user commands over UART/\+Bluetooth and updates shared state. 

Provides helpers for pulling fixed-\/length numeric inputs, changing sensor attributes, and relaying calibration or gain updates to the rest of the system through {\ttfamily task\+\_\+share} queues.
