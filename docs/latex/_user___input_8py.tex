\doxysubsection{Detailed Description}
Bluetooth/serial command handler for Romi. 

Polls UART for user commands, decodes configuration changes, calibration triggers, and direct state transitions, then writes results into shared variables for other tasks. This task is cooperative and yields after each loop iteration.
