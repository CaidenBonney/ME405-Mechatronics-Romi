\doxysubsection{Detailed Description}
Helper class for monitoring the battery voltage on the Romi robot. 

Reads an ADC channel and converts the raw reading into an estimated battery percentage and voltage using nominal calibration values.

The class wraps a {\ttfamily pyb.\+ADC} instance and uses calibrated nominal readings to estimate actual battery values.
