\doxysubsection{Detailed Description}
This file contains a helper class for monitoring the battery voltage on the Romi robot. 

It reads an ADC channel and converts that reading into an estimated battery percentage and voltage.

The class wraps a {\ttfamily pyb.\+ADC} instance and uses calibrated nominal readings to estimate actual battery values.
